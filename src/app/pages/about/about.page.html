Sharp Score

Sharp Score is a tool for playing and tracking a variety of dart games. Players can jump right in to a quick play game, or set up an account and create their own custom game modes.

This application has been built with Angular9 and Ionic, while using Firebase for the backend database and static hosting.

Registration

Users can register either with an exisiting Google account, or with an email and password. Upon becoming registered, users can create/edit a display name for their SharpScore account. This name will be used on the dart game score sheets, and for the time being, can be changed at will.

Using the app

From the home page, non-registered and registered users can use the quick play feature to set up a quick game, that will not be saved to the database. This quick play allows the user to select between several preset game modes.


Registered users will have the option to use the advanced game setup feature. This allows the user to edit more in depth game rules and options, as well as saving games to their account, and viewing the scoresheets of games played in the past.
